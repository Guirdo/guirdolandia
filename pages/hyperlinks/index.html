Marcadores

Páginas interesantes, herramientas, juegos, software, todo lo que se te ocurra al alcance de un click

Streaming

  • Nuclear: Reproductor de música gratuito y sin anuncios. Utiliza fuentes abiertas de todo el internet para traerte la música que te gusta, hasta la puedes descargar!! Solo disponible para computadora.
  • Stremio: Programa que utilizo para ver los torrents que descargo. Hace buen match con la siguiente.
  • yts: Sitio donde descargo torrents para ver peliculas en su idioma original.

Literatura y poesia

  • ebookelo: Biblioteca con miles de ebooks en distintos idiomas pero principalmente en español.
  • Project Gutenberg: Biblioteca en inglés.
  • Tomos Un libro una casa: Colección de libros subidos cada cuando por Un libro Una casa.

Desarrollo de software

  • Codeberg: Repositorio de git en la nube. Alternativa a Github y Gitlab de código abierto y mantenido por la comunidad.

Recursos multimedia

  • GifCities: Biblioteca de gifs de Archive.org

Redes sociales sin algoritmos

  • Linux Pizza (Mastodon): Instancia de mastodon para fans de linux y la tecnologia, pero tambien abierto para cualquiera. Tambien puedes buscar otra instancia que se ajuste a tus necesidades.
